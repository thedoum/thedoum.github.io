Bookmarks Commander – Linux Keyboard Reference

Toolbar button → commander in a new tab. Left pane starts active; use keys below only (Linux / Ctrl).

Basic Navigation & Panels

ActionShortcut
Open commander Click toolbar button once
Move up / down in active pane ↑ / ↓
Open folder / bookmark (same tab) Enter or double‑click
Open bookmark in background tab Ctrl + Enter
Open in background tab (mouse) Ctrl + double‑click
Jump directly by first letter Type first character repeatedly
Parent directory (one level up) Backspace
Toggle between left / right panes Tab
Focus left pane Ctrl + 1 or ←
Focus right pane Ctrl + 2 or →
Reset both panes to root Ctrl + O
Reset active pane to root Ctrl + Shift + O
Active pane: white background. All operations act on it unless stated otherwise. [web:1][web:5]

Selection, Move & Sort

ActionShortcut
Select multiple items Ctrl + left‑click
Add to selection (up/down) Ctrl + ↑ / ↓
Remove from selection (up/down) Ctrl + Shift + ↑/↓
Select all editable entries Ctrl + A
Move selection to right pane Ctrl + →
Move selection to left pane Ctrl + ←
Duplicate selection to other pane Ctrl + U
Move items to top / bottom Alt + Home / End
Move items up / down Alt + ↑ / ↓
Sort A → Z Ctrl + J
Sort Z → A Ctrl + Shift + J
Custom sort / reverse custom Alt + J / Alt + Shift + J

Clipboard & Export

ActionShortcut
Copy links to clipboard Ctrl + C
Copy titles to clipboard Ctrl + X
Copy bookmark IDs to clipboard Ctrl + I
Export selected to clipboard (JSON) Ctrl + Y
Export selected to JSON file Ctrl + Shift + Y
Import from clipboard Ctrl + P
Transfer between browsers: export with Ctrl+Y on source, import with Ctrl+P on target. [web:1]

Create, Edit, Sync & Search

ActionShortcut
New bookmark Ctrl + B
New folder Ctrl + D
Edit title Ctrl + T
Edit URL Ctrl + L
Delete selected item(s) Ctrl + Delete
Mirror inactive pane to active folder Ctrl + M
Mirror inactive pane to selected folder Ctrl + Shift + M
Open selected path in opposite pane Alt + M
Open selected path in current pane Alt + Shift + M
Sync two panes (bookmarks only) Ctrl + Shift + S
Search bookmarks Ctrl + F
Find duplicated bookmarks Ctrl + Shift + F
Duplicates: use duplicates keyword in search bar to list duplicates in current folder + subfolders. [web:1]

Configuration

ActionShortcut
Open configuration / command panel Ctrl + S
From configuration you can change font size, font family, theme (dark/light/OS), and toolbar icon color; you can also toggle single/dual‑pane layout from the command prompt. [web:1][web:5]

Interface Mode (Browser)

  • Right‑click toolbar icon → choose Tab / Window / Popup.
  • Assign a global shortcut in chrome://extensions/shortcuts (or your browser’s equivalent) to open the commander quickly. [web:5]
